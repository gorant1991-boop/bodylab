# BodyLab v4

Интерактивный симулятор воздействий на организм человека. Визуализирует влияние веществ, спорта и питания на органы в реальном времени.

## Что умеет

- 3D-модель тела с анимацией органов (сердце, лёгкие, мозг, печень, почки и др.)
- Drill-down навигация: категория → конкретное вещество/активность
- Дозировка с расчётом воздействия с учётом времени суток и хронических заболеваний
- Вкладки: Вещества (алкоголь, курение, яды, радиация, лекарства), Спорт (кардио, силовая, mind-body), Еда (белки, углеводы, жиры, овощи)
- Биохимические механизмы для каждого вещества
- AI-анализ состояния через Anthropic API
- Предупреждения об опасных комбинациях

## Запуск

Чистый HTML — никакого билда. Просто открыть в браузере:

```bash
open index.html
```

## AI-функция

Кнопка «Проанализировать состояние» отправляет запрос на `https://api.anthropic.com/v1/messages`.  
Работает только при наличии CORS-прокси или из среды где разрешены прямые запросы к Anthropic API.
